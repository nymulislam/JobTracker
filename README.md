## Answers to Questions

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
### Ans.I can get an specific element from html by using getElementById method that have ID, and can get an element getElementsByClassName by classname , and querySelector method allow class, id and tag name and return first matching element / and querySelectorAll returns a node list matching all id, class or tag name.

### 2. How do you create and insert a new element into the DOM?
### Ans. create a new element by using "document.createElement = "[element name]" method and insert a new element into the DOM with append or appendChild method.

### 3. What is Event Bubbling? And how does it work?
### Ans. Event Bubbling is a js concept, when an element receive event, it's bubble up to its parent.

### 4. What is Event Delegation in JavaScript? Why is it useful?
### Ans. Event Delegation is an event handling patter, that allow to handle upper level. It useful for less writing event listener and similar code also.

### 5. What is the difference between preventDefault() and stopPropagation() methods?
### Ans. PreventDefault() is stop default event behavior and stopPropagation() is prevents further propagation from same event.
